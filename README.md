# 🤡 BDA 1. Aprīļa MI Asistente

Šī ir īpaša Streamlit aplikācija, kas izstrādāta BDA MI skolas ietvaros kā pirmā aprīļa pārsteigums. Asistente ir "nedaudz" nogurusi no datiem un mēdz atbildēt visai savdabīgi.

## ✨ Funkcijas
- **Interaktīvs čats:** Reaģē uz Python, SQL un Excel pieminēšanu (bieži vien ar sarkasmu).
- **Balss sintēze:** Asistente runā (izmantojot `edge-tts`), lai kuratorei nebūtu tikai jālasa.
- **Vizuālie efekti:** Konfeti, sniegs un kustīgs fons svētku noskaņai.
- **Īpašā poga:** Liels, zaļš "sprādziens", kas paziņo par brīvdienām.

## 🛠️ Tehnoloģijas
- **Python** (loģika un datu apstrāde)
- **Streamlit** (lietotāja saskarne un CSS maģija)
- **Edge-TTS** (balss ģenerēšana)
